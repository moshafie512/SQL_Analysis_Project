📊 Layoffs Data Analysis

Overview:

This project analyzes a global layoffs dataset using SQL to uncover key trends, patterns, and insights across companies, industries, and countries.

Tools Used:

MySQL

SQL (Joins, Window Functions, Aggregations)


Dataset:

layoffs.csv (global layoffs dataset)


Data Cleaning:

Removed duplicate records using ROW_NUMBER()

Standardized inconsistent values (company, industry, country)

Handled missing values and empty fields

Converted date column to proper DATE format


Key Analysis:

Total layoffs by company

Layoffs distribution by country

Yearly and monthly trends

Top companies with highest layoffs per year


Key Insights:

Layoffs peaked during recent economic downturn periods

The tech industry was the most affected

A small number of companies contributed to a large portion of total layoffs


Skills Demonstrated:

Data Cleaning using SQL

Exploratory Data Analysis (EDA)

Aggregations and Grouping

Basic SQL optimization


Why This Project Matters?

This project demonstrates the ability to transform raw data into meaningful insights using SQL, which is a core skill for any Data Analyst.
